# App Collection

This repository hosts a curated collection of open-source Android apps across various categories:

| Category | Apps |
| :--- | :--- |
| **System** | PojavLauncher, Obtainium, Neo Store, Widgets Pro |
| **Entertainment** | AnymeX, Mihon, CloudStream, Spotube, SimpMusic, Echo |
| **Development** | Termux (Monet, API, Float, Style, X11), NeoTerm, Visual Code Space |
| **Communication** | Signal, Mastodon, Element X |
| **Security** | Stratum (Authenticator) |

## Complete App List

For the complete and up-to-date list of apps in the repository, please see the [apps.yaml](../apps.yaml) file.

## Categories

The repository is organized into the following categories:

- **System**: System utilities, launchers, backup tools, input methods
- **Entertainment**: Media players, manga readers, music apps, video apps
- **Development**: Terminal emulators, code editors, development tools
- **Communication**: Messaging apps, social media clients
- **Security**: Authentication apps, privacy tools
- **Gaming**: Game launchers, emulators
- **Misc**: Various utility applications
