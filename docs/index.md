## Repository Structure

- apps.yaml - Source of truth for apps in the repository
- apks/ - Directory for downloaded APKs (not committed to git)
- fdroid/ - F-Droid configuration and metadata
- scripts/ - Automation scripts for repository management
- website/ - Static website files for repository hosting
- docs/ - Documentation files (this directory)
